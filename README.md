# News-Content-Collect-and-Store
This repository consists of three projects:\
1. The first one **article** is an API develped using Djano framework and BeautifulSoup and newspaper3k libraries the first one is forweb crawlin and the second one is for news crawling and cleaning. In this API a user give the API a news website and the API extracte and clean all the articles and their related information (Authors, Title, Publish Date, The URL of the article that existe in the news website) and put them in a MongoDB database (here MongoDB Atlas).
